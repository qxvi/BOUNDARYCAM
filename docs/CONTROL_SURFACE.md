# Control Surface

BOUNDARYCAM is a public audience and inspection surface.

It is not the evidence source, not the authority source, and not the completion perimeter.

The control surface exposes public objects:

* `public-control.json`
* `boundarycam-manifest.json`
* `data/examples.json`
* `data/surfaces.json`
* `schemas/boundary-frame.schema.json`
