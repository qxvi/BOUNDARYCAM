# BOUNDARYCAM Interactive Boundary Console

BOUNDARYCAM v0.4.0 adds a browser-side Boundary Frame builder.

The console creates a public JSON Boundary Frame from these fields:

- actor
- action
- target
- authority
- execution
- evidence
- replay
- recognition
- recourse
- closure

The console is public-only. It does not send data to a server. It does not claim legal proof. It does not claim private evidence possession.

Primary question: What crossed the boundary?
